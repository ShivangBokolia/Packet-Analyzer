# Packet-Analyzer
Analyzes TCP, UDP and ICMP packets sent over a network.

Compiling the project:
The following java code reads a packet and produces a detailed summary of the packet.
The code takes in a single aargument and that is the packet file itself: <data file>

To compile and execute the program:
1. javac *.java
2. java pktanalyzer <data file>
